## ✨ 소개
"다양한 기술적 시도를 통해 최적의 구현 방법을 탐색하고, 웹 서비스상에서의 실무 적용 가능성을 확인하고자 제작한 프로젝트입니다."


📖 개요
개발 기간: 2026.03 ~ 



## 🛠 주요 기능
* **사용자 인증 및 관리 (Auth)**: Session 기반의 보안 로그인 및 회원가입 기능 구현, 유효성 검사(Validation)를 통한 사용자 데이터 무결성 확보
* **커뮤니티 게시판 (CRUD)**: 게시글 작성, 조회, 수정, 삭제의 완전한 CRUD 로직 구현, RESTful API 설계 원칙을 준수하여 프론트엔드-백엔드 간 효율적인 데이터 통신 수행
